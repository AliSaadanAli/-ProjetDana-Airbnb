# Projet Dana – Airbnb (Groupe 10)
Dataset Airbnb (10k annonces) pour le projet Web Sémantique.

## Description
Ce dépôt contient un sous-ensemble de 10 000 annonces Airbnb issues du site open data [Inside Airbnb](https://insideairbnb.com/get-the-data/), pour la ville de Paris.

Les données ont été converties en RDF dans le cadre du projet **Dana / Web Sémantique**.

## Fichiers
- `paris_airbnb.csv` → données sources
- `airbnb.ttl` → version RDF du dataset
